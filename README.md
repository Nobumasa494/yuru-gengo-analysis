# ゆる言語学ラジオ YouTube動画分析

YouTubeチャンネル「ゆる言語学ラジオ」の全488本の動画データを収集・分析し、  
**どんなタイトル・テーマが再生数に影響するか**を統計・機械学習で検証したポートフォリオプロジェクト。

---

## 分析の目的

- タイトルのテーマや構成が再生数に影響するかを統計的に検証する
- 機械学習で「人気動画かどうか」を予測できるか試みる
- データに基づいたコンテンツ戦略の示唆を得る

## 分析の流れ

### 1. データ収集
YouTube Data API v3を使って全488本の動画情報（再生数・いいね数・コメント数・投稿日時）を取得。

### 2. 特徴量抽出
タイトルのキーワードをもとに、以下5つの特徴量をルールベースで付与。

| 特徴量 | 内容 |
|---|---|
| theme | 動画テーマ（クイズ・ゲーム、語源・語彙 など14カテゴリ） |
| emotion | タイトルが誘う感情（驚き、疑問、知識欲 など） |
| difficulty | 難易度（初級・中級・上級） |
| audience | 想定視聴者（言語好き、雑学好き など） |
| title_type | タイトルの型（質問型、クイズ型、断定型 など） |

### 3. 統計検定
Kruskal-Wallis検定 → Dunn検定（Bonferroni補正）で各特徴量と再生数の関係を検証。

### 4. 機械学習
RandomForest・LightGBM・CatBoostのアンサンブルで「人気動画かどうか」を二値分類。

---

## 主な発見

### 統計分析

![テーマ別 中央値再生数](images/theme_views.png)

- **クイズ・ゲームテーマ**の中央値再生数は約34万回で、最下位「その他」（15万回）の2倍以上（p<0.001）
- **クイズ型タイトル**は断定型・質問型より有意に高い（p<0.001）。「〇〇はどっち？」など視聴者参加型の問いかけが効果的
- テーマの影響（H=70.95）はタイトル型（H=33.09）より大きく、**「何について話すか」が「どう言うか」より重要**
- 難易度では初級（クイズ・雑談）が上級より高く、専門性よりとっつきやすさが再生数に寄与している

### 機械学習

- RandomForestの精度は**55%**（ランダム比+5%）にとどまり、タイトル特徴だけでは予測が困難
- 特徴量を追加（投稿曜日・シリーズ番号）しても**59%止まり**。シリーズ番号が重要特徴量になったが、これはチャンネル成長の代理変数であり汎用性がない
- LightGBM・CatBoostとのアンサンブルでも改善なし

### 考察

統計分析では**クイズ・ゲーム系のコンテンツが有意に高い再生数**と関連することが示された。  
一方で機械学習の精度が低かった理由は、再生数がサムネイル・投稿タイミング・チャンネル登録者数の増加など**タイトル以外の要因に大きく左右されるため**と考えられる。

統計的知見はコンテンツ戦略の参考になるが、クイズ動画に偏りすぎるとチャンネルの世界観が崩れるリスクもある。  
**「何を作るか」の判断材料としてデータを活用しつつ、最終的にはコンテンツの質で勝負する**というバランスが重要だと考える。
